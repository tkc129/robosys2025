# robosys2025
ロボットシステム学の課題用ファイルをまとめたリポジトリです。

## 概要
標準入力から 「時給」「1日の勤務時間」「月の勤務日数」 の3つの数値を受け取り、日給・月給・年収・税金を計算し、標準出力するコマンドです。

## 実行環境
以下の環境、ソフトウェアにおいてプログラムの実行及びテストを確認しています。
- Ubuntu 22.04.5 LTS
- python 3.8～3.12

## 実行方法
"git"コマンドを使用してリポジトリをクローンし、robosys2025に移動する事で実行できます。
下記のようにecho の後に任意に数字を入力してください。
入力された値のそれぞれの計算結果が出力されます。
```
$ git clone https://github.com/tkc129/robosys2025.git
$ cd robosys2025
$ echo "時給 1日の勤務時間 月の勤務日数" | ./calcwage
```

## 税率
所得税の税率は以下の表通りに出力されます。また、住民税の税率は一律で10％です。
例外として、年収が123万円以下の場合は所得税が0%となり、年収が135万円以下の場合は住民税が0%になります。
|所得金額|税率|
|----|----|
|195万円以下|5%|
|330万円以下|10%|
|695万円以下|20%|
|900万円以下|23%|
|1800万円以下|33%|
|4000万円以下|40%|
|4000万円超|45%|


## 実行例
時給1,200円、1日8時間、月に20日勤務と仮定した場合
```
$ echo "1200 8 20" | ./calcwage
額面給与: 日給 9,600円 / 月給 192,000円 / 年収 2,304,000円
適用税率: 所得税 10% + 住民税 10% (計 20%)
推定手取り年収: 1,843,200円
```
